# Flutter Dev Camp News App

News Application built as one of the assignments during the [Flutter Dev Camp](https://www.flutterdevcamp.com/).

## News API Source

The News data shown here is from [GNews](https://gnews.io/). GNews API is a simple REST API with which you can search for current and historic news articles published by over 60,000 sources.
